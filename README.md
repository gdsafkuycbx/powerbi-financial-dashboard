# powerbi-financial-dashboard
This project presents an interactive financial dashboard built using Power BI to analyze income, expenses, and profitability trends.
🎯 Objectives
Track monthly financial performance
Analyze income vs expenses
Identify spending patterns
Forecast future financial trends
🛠️ Tools & Technologies
Power BI
Excel
DAX (Data Analysis Expressions)
📸 Dashboard Preview

📊 Key Metrics
Total Income
Total Expense
Profit
Profit Margin
